## 📂 Project Structure

```
ALGORITHMS/
├── BACKTRACKING_ALGORITHMS/     # N-Queens, Sudoku Solver
├── DYNAMIC_PROGRAMMING/          # DP problems (Knapsack, LCS, Fibonacci, etc.)
├── GRAPH_ALGORITHMS/             # BFS, DFS, Dijkstra, Topological Sort, etc.
├── GREEDY_ALGORITHMS/            # Greedy approach problems
├── MATHEMATICAL_ALGORITHMS/      # GCD, Modular Exponentiation, Sieve
├── SEARCHING_ALGORITHMS/         # Binary, Linear, Jump, Interpolation Search
├── SORTING_ALGORITHMS/           # Quick, Merge, Heap, Bubble, etc.
└── TREE_ALGORITHMS/              # BST, Tree Traversals, LCA
```

## 🏷️ Algorithm Categories

| Category | Description | Files |
|----------|-------------|-------|
| **Backtracking** | Solve problems by trying possibilities and backtracking | 2 |
| **Dynamic Programming** | Optimize problems by storing subproblem solutions | 6 |
| **Graph Algorithms** | BFS, DFS, Dijkstra, Union-Find, Topological Sort | 9+ |
| **Greedy Algorithms** | Make locally optimal choices | 1 |
| **Mathematical** | GCD, Prime numbers, Modular arithmetic | 3 |
| **Searching** | Binary, Linear, Jump, Interpolation search | 6 |
| **Sorting** | Quick, Merge, Heap, Bubble, Insertion, Selection | 7 |
| **Tree Algorithms** | BST operations, Traversals, LCA | 3 |

## 📊 Complexity Cheatsheet

| Algorithm | Time Complexity | Space Complexity |
|-----------|-----------------|------------------|
| Binary Search | O(log n) | O(1) |
| Merge Sort | O(n log n) | O(n) |
| Quick Sort | O(n log n) avg | O(log n) |
| Dijkstra | O((V+E) log V) | O(V) |
| BFS/DFS | O(V+E) | O(V) |
| Knapsack (DP) | O(nW) | O(nW) |

## 🛠️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/Aritradutta2002/Algorithms.git
cd Algorithms
```

### Compile & Run (Java)

```bash
# Example: Run Binary Search
cd SEARCHING_ALGORITHMS
javac BinarySearch.java
java BinarySearch
```
